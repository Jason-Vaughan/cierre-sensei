# Privacy Policy (for Cierre Sensei)

**Last updated:** December 2025

Cierre Sensei is a read-only closing-cost estimation tool for residential real-estate purchases in Mexico.

It does not collect, store, or share any personal information outside of what is processed by OpenAI during the GPT interaction.

---

## 1. What data we access

Cierre Sensei only performs one external request:

- A read-only GET request to fetch a publicly available JSON dataset hosted at GitHub:

  https://raw.githubusercontent.com/Jason-Vaughan/cierre-sensei/main/cierre_sensei_data_json

**No user identifiers, personal information, or query contents are sent to GitHub.**

---

## 2. What we do NOT collect

Cierre Sensei does not:

- Collect user data
- Store user inputs
- Track sessions
- Use cookies
- Log IP addresses
- Share or sell user information

---

## 3. Third-party services

This GPT uses:

- **OpenAI** (for chat processing)
- **GitHub Raw Content** (for loading the public JSON dataset)

These services may have their own privacy policies.

---

## 4. Security

The GPT and dataset are read-only.

**No write operations or user data storage occur.**
